2. Dados Coletados
O app coleta apenas o que você insere voluntariamente:
✔ Nomes dos moradores
✔ Fotos dos moradores (opcional)
✔ Valores e descrições de contas
✔ Senha de acesso
NÃO coletamos: localização, e-mail, CPF, contatos ou dados bancários.
3. Finalidade
Os dados são usados exclusivamente para o funcionamento do app. Não são compartilhados com terceiros nem usados para publicidade.
4. Armazenamento
Dados armazenados no Firebase (Google Cloud), com certificações de segurança internacionais. Cada casa tem código único e dados completamente isolados.
5. Seus Direitos (LGPD)
✔ Acessar seus dados diretamente no app
✔ Corrigir ou atualizar a qualquer momento
✔ Apagar via botão "Apagar histórico"
✔ Solicitar exclusão total via WhatsApp
6. Sua Responsabilidade
Você é responsável pelos dados que insere, incluindo fotos e nomes de terceiros. Ao cadastrar dados de outras pessoas, você declara ter o consentimento delas.
7. Retenção
Dados ficam armazenados enquanto a casa estiver ativa. Casas sem acesso por mais de 12 meses podem ser removidas mediante aviso prévio.
